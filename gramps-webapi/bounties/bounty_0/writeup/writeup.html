CVE-2026-40258: Zip Slip Path Traversal in Media Archive Import

Severity: Critical (CVSS 9.8)

CWE: CWE-22 - Path Traversal

Affected: gramps-webapi

Description

The media archive import functionality in gramps-webapi is vulnerable to a Zip Slip attack. When importing a media archive (zip file), the application extracts files without validating that the extracted file paths stay within the intended extraction directory. An attacker can craft a malicious zip archive containing entries with path traversal sequences (e.g., ../../etc/crontab) to write arbitrary files anywhere on the filesystem.

Impact

An attacker can write arbitrary files to the server filesystem, potentially achieving remote code execution by overwriting application files, cron jobs, or SSH keys.

Fix

The fix validates that extracted file paths resolve to within the temporary extraction directory before writing, preventing path traversal.
