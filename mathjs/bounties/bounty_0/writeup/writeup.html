GHSA-jvff-x2qm-6286: mathjs Arbitrary Code Execution via Expression Parser

Overview

A high-severity vulnerability (CVSS 8.8) was discovered in mathjs that allowed executing arbitrary JavaScript via the expression parser.

Affected Package

  • Package: npm mathjs
  • Vulnerable versions: Before 15.2.0
  • Fixed version: 15.2.0

Vulnerability Description

Two security vulnerabilities were detected that allowed executing arbitrary JavaScript via the expression parser of mathjs.

Users are at risk if their applications permit users to evaluate arbitrary expressions using the mathjs expression parser.

Severity

  • CVSS v3 Score: 8.8 / 10 (High)
  • Attack Vector: Network
  • Attack Complexity: Low
  • Privileges Required: Low
  • User Interaction: None
  • Impact: High confidentiality, integrity, and availability

Weakness

CWE-915: Improperly Controlled Modification of Dynamically-Determined Object Attributes

Remediation

There is no workaround without upgrading. Users must update to version 15.2.0 or later.
